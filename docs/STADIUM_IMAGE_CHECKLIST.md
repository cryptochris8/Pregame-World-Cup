## Quick Download Guide

### Method 1: Unsplash (Recommended - Free)
1. Go to https://unsplash.com/
2. Search for stadium name
3. Find high-res image (look for "Free to use" label)
4. Click image → Download → Choose "Large" or "Original"
5. Rename to match filename above
6. Save to `assets/worldcup/stadiums/`

### Method 2: Official Stadium Websites
1. Visit stadium's official website
2. Look for "Media", "Press", or "Gallery" sections
3. Download high-resolution images
4. Check licensing terms (most allow editorial use)

### Method 3: Wikimedia Commons
1. Go to https://commons.wikimedia.org/
2. Search: "Stadium Name"
3. Filter by resolution (look for 1920x1080 or higher)
4. Check license (CC-BY-SA or Public Domain preferred)
5. Download and rename

## Image Quality Requirements

✅ **Required Specifications:**
- Format: JPG or PNG
- Minimum Size: 1920x1080px (16:9 aspect ratio)
- Quality: High resolution, sharp, professional
- View: Exterior or interior panoramic view
- Lighting: Well-lit, clear visibility
- Naming: Lowercase with underscores

❌ **Avoid:**
- Low resolution or blurry images
- Images with heavy watermarks
- Night shots with poor lighting (unless exceptional)
- Images with obstruction or poor framing

## Image Optimization (Optional)

If images are too large, compress them:

### Using ImageMagick (Command Line)
```bash
cd assets/worldcup/stadiums
for file in *.jpg; do
  convert "$file" -resize 1920x1080 -quality 85 "optimized_$file"
done
```

### Using Online Tools
- TinyJPG: https://tinyjpg.com/ (drag & drop, auto compress)
- Squoosh: https://squoosh.app/ (Google's image compressor)
